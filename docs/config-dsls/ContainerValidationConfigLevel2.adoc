You need this when your container needs to peek 2 levels deep for validations relation to its batch members or containers it contains.

== Example

ifdef::env-github[]

[source,java,indent=0]
----
class Bean {}

class Header3 {
  List<Bean> beanBatch;
}

class Header2 {
  List<Header3> header3Batch;
}

class Header1Root {
  List<Header2> header2;
}
----

endif::[]
ifndef::env-github[]

----

----

endif::[]
